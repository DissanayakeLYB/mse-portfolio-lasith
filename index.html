Image Analysis for Fish Embryo Acute Toxicity (FET) Tests

2026 (Ongoing)

Developing an image analysis pipeline using ImageNet, ResNet, SAM, and YOLO models to analyze embryo images at 24h, 48h, and 72h. Test samples are repeated across composition ratios (1:1, 1:2, 2:1) and concentration levels (e.g., 225 ppm:225 ppm, 450 ppm:450 ppm, 900 ppm:900 ppm) to compare toxicity-induced damage and growth deviations against control experiments.

Computer Vision ImageNet/ResNet SAM YOLO

Machine Learning Based Optimization of CZTS Photovoltaic Parameters

2025

Related six photovoltaic output parameters (including short-circuit current density and open-circuit voltage) with selected input variables using machine learning. Built non-linear empirical equations for each output, validated them against SCAPS-1D simulations, and derived a second-degree bandgap equation as a function of Cu, Zn, and Sn compositional ratios.

Machine Learning Photovoltaics CZTS SCAPS-1D
